⎯ Descripción de Guías ⎯

Guía 1

Los ejercicios de la Guía 1 se centran en aplicar estilos CSS de diferentes maneras: embebiendo estilos directamente en etiquetas HTML, usando la etiqueta style dentro del documento, y finalmente utilizando un archivo CSS externo para mantener un diseño más organizado. Se pide a los estudiantes crear varias páginas HTML donde aplicarán colores, tamaños de fuente, alineación y otros estilos, mientras que en los ejercicios complementarios deben crear una página sobre su programa favorito y publicar los ejemplos en GitHub Pages.

Guía 2

En la Guía 2, los ejercicios avanzan hacia el uso de selectores CSS más complejos y el modelo de caja. El primer ejercicio involucra estilizar una página HTML usando selectores avanzados, pseudo-clases y pseudo-elementos. El segundo ejercicio reta a los estudiantes a diseñar un tablero de damas aplicando la pseudo-clase :nth-child(). Como ejercicios complementarios, se les pide crear una biografía personal con estilos avanzados y experimentar con Google Fonts para personalizar la tipografía.

Guía 3

Los ejercicios se enfocan en aplicar técnicas de posicionamiento y visualización CSS para crear un diseño de sitio web. El primer ejercicio consiste en construir una página sobre nominaciones a los Óscares, usando diferentes tipos de posicionamiento (relativo, absoluto, y flotante) para organizar los elementos en la pantalla. También se deben aplicar propiedades como display, overflow y z-index para controlar la visualización de los elementos. El segundo ejercicio implica diseñar un menú horizontal multinivel utilizando listas y estilos CSS. Finalmente,el tercer ejercicio trata de construir un diseño fijo para una página de una agencia de viajes.

Guía 4

Esta guía tiene ejercicios enfocados en la creación y estilización de formularios HTML5, así como la implementación de diseño responsivo. El primer ejercicio consiste en construir un formulario alineado sin usar tablas, utilizando varios tipos de controles HTML5. El segundo ejercicio explora más controles de formularios, y el tercero se centra en enviar la información. El cuarto ejercicio es sobre diseñar un formulario de login con estilos CSS. Finalmente, el quinto ejercicio introduce las media queries para hacer un sitio responsivo, y el sexto implica ajustar un diseño de una agencia de viajes para adaptarse a diferentes dispositivos.

Desafío Práctico

El desafío consiste en construir un sitio web para la Bolsa de Trabajo de ESEN utilizando HTML y CSS, con cuatro páginas: index.html, ofertas.html, aplicar.html y consejos.html. El objetivo es crear un diseño responsivo siguiendo una imagen de referencia proporcionada. En la página ofertas.html, se debe presentar una tabla con las ofertas de empleo vigentes. En aplicar.html, se requiere un formulario para que los estudiantes puedan aplicar a las ofertas, asegurando que los datos se envíen a aplicar.php sin aparecer en la URL. Finalmente, en consejos.html, se debe insertar un video con una imagen de portada hasta que se inicie su reproducción.
